A simple and very minimal notice bar showing plugin

== Description ==

The Announcement Notice is a very tiny and easy to use WordPress plugin based on the Elementor. It helps you to show your notice on the sticky header. You can build/edit the notice bar with the Elementor plugin.


== Installation ==

1. Upload the entire `announcement-notice` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Announcement Notice Settings** section in Announcement > All Notices
